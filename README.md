🚀Convite Digital Interativo - Projeto UX/UI
Este projeto foi desenvolvido para transformar um convite tradicional em uma interface funcional e automatizada, melhorando a experiência do usuário (UX) e a logística de eventos.

🛠️ Funcionalidades Técnicas:
Integração RSVP: Botão conectado ao Google Forms para coleta de dados em tempo real, gerando automaticamente uma planilha no Google Sheets para o organizador.

Geolocalização: Link direto para o Google Maps facilitando a navegação dos convidados.

Gestão de Agenda: Inclusão do evento no calendário do smartphone via link do Google Agenda.

Suporte via WhatsApp: Botões com mensagens pré-configuradas para dúvidas e sugestões de presentes.

💻 Ferramentas Utilizadas:
Design: Canva (Interface e Prototipagem).

Back-end de Dados: Google Forms & Sheets.

Versionamento: GitHub.
